# 🌿 Reflections on Religion & Life — تأملات في الدين والحياة

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **تأملات في الدين والحياة (Reflections on Religion and Life)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on life's purpose, death, happiness, suffering, nature, worship, gratitude, family, knowledge, balance
- 6 reflections with deep insight format
- Daily habit tracker, self-assessment quiz, duas
- Trilingual: Arabic, English, French with RTL/LTR
- 3 themes (Teal, Night, Garden), Narrator, PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.

## Disclaimer

I am not a scholar. This is a humble effort by a Muslim parent to share Sheikh al-Ghazali's wisdom interactively. Content derived from the book and trusted Islamic sources. This is not a fatwa.
